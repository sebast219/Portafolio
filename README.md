# Portafolio

Proyecto de portafolio personal con **React** y **Tailwind CSS**, creado con Vite.

## Requisitos

- Node.js 18+

## Cómo usar

```bash
# Instalar dependencias
npm install

# Modo desarrollo
npm run dev

# Compilar para producción
npm run build

# Vista previa de la build
npm run preview
```

## Estructura

- `src/App.jsx` — Componente principal que une todas las secciones
- `src/components/` — Header, Hero, About, Projects, Skills, Contact, Footer
- `src/index.css` — Estilos globales y directivas de Tailwind

Personaliza textos, enlaces, proyectos y habilidades en cada componente.
